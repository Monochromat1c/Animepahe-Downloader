- To download, open git bash inside animepahe-dl directory
- Run the command: ./animepahe-dl.sh
- Open the anime.list file inside the directory
- Look for the anime you want to download and copy the unique key inside the brackets
- Inside the animepahe-dl folder, open Git
- Run the command: ./run.sh
- Enter the key, episode/range of episodes you want to download, and the resolution
